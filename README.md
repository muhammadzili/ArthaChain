## ğŸš€ Cara Penggunaan

### Persyaratan Sistem

- Python 3.8+
- pip
- Git

### Instalasi

1. Clone repository:
   ```bash
   git clone https://github.com/muhammadzili/ArthaChain.git
   cd ArthaChain
   ```

2. Buat virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate      # Linux/MacOS
   venv\Scripts\activate       # Windows
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
